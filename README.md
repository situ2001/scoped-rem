# scoped-rem

| Package                           | Description                                   |
| --------------------------------- | --------------------------------------------- |
| [scoped-rem](./lib/core)          | Bundler-agnostic library for transforming CSS |
| [scoped-rem-loader](./lib/loader) | Webpack loader for `scoped-rem`               |
